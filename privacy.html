Privacy Policy

นโยบายความเป็นส่วนตัว

อ่านสลิปถูกออกแบบให้ผู้ใช้ควบคุมข้อมูลสลิปได้ชัดเจน

ข้อมูลที่แอปประมวลผล

  • รูปสลิปที่คุณถ่ายหรือเลือกจากคลังรูป
  • ข้อมูลที่อ่านหรือแก้ไขจากสลิป เช่น ยอดเงิน วันที่ ผู้รับ เลขอ้างอิง หมวดหมู่ และหมายเหตุ
  • ข้อมูลร้านในโหมดร้าน เช่น ชื่อร้าน บทบาทผู้ใช้งาน พนักงาน ร้านที่เจ้าของสร้าง และรายการที่ส่งเข้าแดชบอร์ดร้าน
  • สถานะสิทธิ์ ReadSlip Pro จาก StoreKit เพื่อเปิดใช้ฟังก์ชันเจ้าของร้าน

OCR และรูปสลิป

ในโหมดส่วนตัว แอปใช้ OCR บน iPhone เพื่ออ่านข้อความจากรูปสลิป และไม่ส่งรูปขึ้นคลาวด์เพื่ออ่าน OCR ข้อมูลและรูปที่บันทึกจะอยู่บนอุปกรณ์ของคุณจนกว่าคุณจะลบเอง

ในโหมดร้าน แอปใช้ Firebase เพื่อซิงก์ข้อมูลร้านข้ามเครื่อง รวมถึงส่งรูปสลิปที่พนักงานหรือเจ้าของร้านบันทึกไปยัง Firebase Storage เพื่อให้เจ้าของร้านตรวจดูจากเครื่องของตัวเองได้

Firebase และ Analytics

โหมดร้านใช้ Firebase Authentication, Cloud Firestore และ Firebase Storage เพื่อจัดการร้าน พนักงาน รายการ และรูปสลิปที่ส่งเข้าแดชบอร์ดร้าน

แอปอาจใช้ Firebase Analytics เพื่อดูเหตุการณ์การใช้งานโดยรวม เช่น เปิดแอป จบ onboarding เริ่มสแกน บันทึกรายการ ซิงก์ร้าน เปลี่ยนช่วงแดชบอร์ด ส่งออกรายงาน PDF/PNG การเรียก native App Store review prompt และสถานะทั่วไปของ paywall/subscription ข้อมูล analytics ไม่ควรมีจำนวนเงิน ชื่อผู้รับ เลขอ้างอิง OCR raw text shop ID หรือ path รูปสลิป

ReadSlip Pro และการซื้อในแอป

ฟังก์ชันเจ้าของร้านใช้ ReadSlip Pro ผ่านระบบซื้อในแอปของ Apple โดยเริ่มทดลองใช้ฟรี 7 วัน แล้วต่ออายุรายเดือน ผู้ใช้สามารถจัดการหรือยกเลิก subscription ได้ใน App Store แอปใช้ StoreKit เพื่อตรวจสิทธิ์และไม่บันทึก transaction ID หรือข้อมูลใบเสร็จลง Firebase Analytics

สิทธิ์ของอุปกรณ์

  • Camera: ใช้ถ่ายรูปสลิป
  • Photos: ใช้เลือกรูปสลิปหรือค้นหารูปภาพล่าสุดที่อาจเป็นสลิป

การลบข้อมูล

ผู้ใช้สามารถลบข้อมูลในเครื่องจากหน้า Settings ได้ เจ้าของร้านสามารถลบร้านที่ตัวเองสร้างได้จากโหมดร้าน เมื่อใช้ Firebase ข้อมูลที่ถูกซิงก์อาจอยู่ใน Firebase จนกว่าจะถูกลบตามฟังก์ชันของแอปหรือนโยบายของผู้ให้บริการ

ข้อจำกัด

อ่านสลิปไม่ใช่บริการธนาคาร และไม่ได้ตรวจสอบว่าสลิปเป็นของจริงกับธนาคาร ผู้ใช้ควรตรวจข้อมูลก่อนบันทึกหรือใช้งานต่อเสมอ

English Summary

Personal mode processes slip OCR on device. Shop mode syncs submitted shop records and slip images through Firebase so the owner can review staff submissions. ReadSlip is not a bank and does not verify slips with banks.
